## ğŸš€ Getting Started

### Prerequisites

### Installation

1. **Clone the repository**
   \`\`\`bash
   git clone https://github.com/tessyjonburica/valentino-ai.git
   cd valentino-ai
   \`\`\`

2. **Install dependencies**
   \`\`\`bash
